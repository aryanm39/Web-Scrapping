This is an A.P. in which a = 6, d = 6 and Sn = 1800

3n (n + 1) = 1800

n(n + 1) = 600

n2 + n - 600 = 0

n2 + 25n - 24n - 600 = 0

n(n + 25) - 24(n + 25) = 0

(n + 25)(n - 24) = 0

n = 24
